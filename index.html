Anotações de Estudo: Citações e Códigos em HTML

1. Introdução

Esta aula aborda tags HTML utilizadas para lidar com citações e códigos de programação. Elas trazem semântica e estrutura ao conteúdo, além de melhorar a acessibilidade e organização.

2. Tags para Citações

<blockquote> - Citação em Bloco

Usado para citações longas retiradas de outras fontes. Pode ter o atributo cite com a URL da fonte.

No livro "1984", George Orwell escreveu:

O Grande Irmão está te observando.

Guerra é Paz. Liberdade é Escravidão. Ignorância é Força.

<q> - Citação Curta Inline

Para citações curtas dentro do texto.

Sócrates disse: Só sei que nada sei.

<cite> - Título da Fonte

Usado para indicar o título de uma obra ou fonte.

Como dito em O Pequeno Príncipe, "Tu te tornas eternamente responsável por aquilo que cativas".

3. Tags para Códigos de Programação

<code> - Código Inline

Para pequenos trechos de código.

Use a função console.log() para imprimir no console.

<pre><code> - Bloco de Código

Para blocos de código com preservação de formatação.


function greet(name) {
    if (name) {
        console.log("Hello, " + name + "!");
    } else {
        console.log("Hello, stranger!");
    }
}

<samp> - Saída de Exemplo

Mostra um exemplo de saída do sistema.

Ao executar o comando, você verá: Hello World!

<kbd> - Entrada de Teclado

Indica entradas feitas pelo usuário no teclado.

Pressione Ctrl + C para copiar.

<var> - Variável

Representa variáveis em programação ou matemática.

A equação é E = mc2.

4. Importância do Uso Semântico

  • Clareza e Contexto: Define claramente o tipo de conteúdo.
  • Acessibilidade: Leitores de tela identificam melhor o conteúdo.
  • SEO: Conteúdo mais bem interpretado pelos buscadores.
  • Manutenibilidade: Código mais limpo e organizado.

Conclusão

As tags de citação e código enriquecem o conteúdo HTML, tornando-o mais significativo e acessível.
